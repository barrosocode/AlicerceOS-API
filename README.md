# Alicerce OS Engine - Core API 🚀

O **Alicerce OS Engine** é o cérebro do sistema de gestão da JELL. Esta API centraliza a inteligência de negócios, controle de fluxo de caixa e o motor de monitoramento de licenciamentos (IDEMA/Bombeiros/Prefeitura).

### 🛠️ Stack Tecnológica

- **Linguagem:** Python 3.12+
- **Framework:** Django & Django REST Framework (DRF)
- **Banco de Dados:** PostgreSQL
- **Autenticação:** JWT (SimpleJWT)
- **Metodologia:** SDD (Spec Driven Development) + TDD (Test Driven Development)

### 🎯 Funcionalidades Chave

- Gestão centralizada de orçamentos e custos de obras.
- Monitoramento inteligente de prazos de licenciamento.
- Fluxo de caixa segregado por projeto e categorias operacionais.
- Engine para futuros serviços de Webscraping.
